# Guía de Actualización del README.md

Esta guía describe cómo hemos transformado el `README.md` original en una versión personalizada para **AutoToDo**, centrada en sus nuevas capacidades de automatización.

## Pasos Realizados

1.  **Identificación de Funcionalidades**: Hemos analizado el código fuente (`src/webview.ts` y `media/main.js`) para identificar los parámetros del motor de automatización: impacto, dificultad, tiempo y fechas.
2.  **Redacción del Aviso de Fork**: Se ha colocado en la primera línea el aviso de que este proyecto es un fork, respetando la autoría original pero aclarando las nuevas adiciones.
3.  **Localización al Español**: Todo el contenido explicativo, técnico y de usuario ha sido traducido al español para mejorar la accesibilidad del usuario actual.
4.  **Sección de Automatización**: Se ha añadido una sección dedicada a "Priorización Inteligente (Auto Priority)" que explica cómo el algoritmo ayuda a decidir qué tarea realizar primero.
5.  **Preservación de Multimedia**: Se han mantenido los enlaces a las capturas de pantalla para asegurar que la presentación visual siga siendo atractiva.

Una vez aplicado, el `README.md` reflejará fielmente el estado actual del proyecto **AutoToDo**.
